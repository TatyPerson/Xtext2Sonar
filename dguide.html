Xtext2Sonar

This project performs the transformation of DSL grammar defined with the Xtext format to the grammar used for the SonarQube platform. This transformation is based on M2T approach.

Developer's guide

Works with Xtext2Sonar

1. Clone the GitHub repository of Xtext2Sonar.

To clone the GitHub repository you can access:

https://github.com/TatyPerson/Xtext2Sonar


2.Modify the generator of the SonarQube plug-in architecture (launcher project).

In this case, you must work with "Xtext2SonarM2T.launcher" project. The class that performs the generation of the structure is "AcceleoGenerateGenerateSonarQubePluginAction.java".


3. Modify the transformer of the Xtext grammar to SonarQube grammar.

In this case, you must work with "Xtext2SonarM2T" project. The file that performs the conversion of the grammar is "generateSonarGrammar.mtl".
